# CinePrompt

**将电影截图拆解为可复刻的 AI 图像生成提示词**

CinePrompt 是一个节点画布工具，上传电影截图后，通过 AI 视觉模型自动拆解画面的空间层次、灯光、摄影机/镜头匹配、材质调色等维度，输出三级精度的结构化提示词，可直接用于 Midjourney、Flux、即梦等 AI 图像生成工具。

## 特性

- **节点工作流画布** — 无限平移/缩放，拖拽式交互，分析结果以节点 + 连线呈现
- **三支柱分析** — 构图几何 · 灯光测光 · 光学传感器规格
- **空间层次拆解** — 前景/中景/背景分层 + 左/中/右分区描述
- **摄影机 & 镜头匹配** — 内置 15 台机身 + 15 组镜头知识库，自动生成 `shot on` 标签
- **三级提示词输出** — L1 精简 / L2 标准 / L3 专业，复制即用
- **风格库** — 按分类（摄影机、灯光、色调、氛围、完整提示词）保存/导入/导出提示词片段
- **节点可调节大小** — 拖拽右下角手柄自由缩放节点尺寸
- **双 API 支持** — Google Gemini / BytePlus (豆包)
- **零依赖单文件** — 一个 HTML 文件，本地双击即可使用

## 快速开始

1. **下载** `index.html` 到本地
2. **双击打开**（或拖入浏览器）
3. 点击右上角 **⚙ API** 配置你的 API Key：
   - **Gemini**: 前往 [Google AI Studio](https://aistudio.google.com/apikey) 获取
   - **BytePlus**: 使用豆包 API 的 Key + Endpoint
4. 上传电影截图 → 点击「拆解镜头」→ 等待节点生成

## 技术栈

- 纯前端，无构建步骤，无框架依赖
- HTML + CSS + Vanilla JS（单文件内联）
- Google Fonts (Inter + JetBrains Mono)
- localStorage 持久化配置与风格库

## 风格库

风格库支持按 5 个分类管理提示词片段：

| 分类 | 用途 |
|------|------|
| 📷 摄影机/镜头 | `shot on` 标签、机身/镜头组合 |
| 💡 灯光 | 光源方向、色温、光质描述 |
| 🎨 色调 | 调色风格、色彩倾向 |
| 🌫 氛围 | 雾气、颗粒、氛围描述 |
| 📋 完整提示词 | 完整的 L1/L2/L3 提示词 |

支持 JSON 导入/导出，方便跨设备迁移。
